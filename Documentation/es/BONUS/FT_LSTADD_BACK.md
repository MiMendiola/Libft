# FT_LSTADD_BACK

---

## SYNOPSIS

```c
#include <libft.h>

void ft_lstadd_back(t_list **lst, t_list *new);
```

---

## DESCRIPTION

Añade el nodo ’new’ al final de la lista ’lst’.

---

## PARAMETERS

lst: el puntero al primer nodo de una lista.
new: el puntero a un nodo que añadir a la lista.

---

## RETURN VALUES

Nada

---

## AUTHORIZED FUNCTIONS

Ninguna
