# FT_PUTSTR_FD

---

## SYNOPSIS

```c
#include <libft.h>

void ft_putstr_fd(char *s, int fd);
```

---

## DESCRIPTION

Envía la string ’s’ al file descriptor especificado.

---

## PARAMETERS

s: La string a enviar.
fd: El file descriptor sobre el que escribir.

---

## RETURN VALUES

Nada
