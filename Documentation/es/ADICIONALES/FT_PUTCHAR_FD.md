# FT_PUTCHAR_FD

---

## SYNOPSIS

```c
#include <libft.h>

void ft_putchar_fd(char c, int fd);
```

---

## DESCRIPTION

Envía el carácter ’c’ al file descriptor especificado.

---

## PARAMETERS

c: El carácter a enviar.
fd: El file descriptor sobre el que escribir.

---

## RETURN VALUES

Nada
