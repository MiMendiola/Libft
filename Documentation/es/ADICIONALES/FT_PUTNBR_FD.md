# FT_PUTNBR_FD

---

## SYNOPSIS

```c
#include <libft.h>

void ft_putnbr_fd(int n, int fd);
```

---

## DESCRIPTION

Envía el número ’n’ al file descriptor dado.

---

## PARAMETERS

n: El número que enviar.
fd: El file descriptor sobre el que escribir.

---

## RETURN VALUES

Nada
