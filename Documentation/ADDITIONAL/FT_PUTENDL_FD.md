# FT_PUTENDL_FD

---

## SYNOPSIS

```c
#include <libft.h>

void ft_putendl_fd(char *s, int fd);
```

---

## DESCRIPTION

Envía la string ’s’ al file descriptor dado, seguido de un salto de línea.

---

## PARAMETERS

s: La string a enviar.
fd: El file descriptor sobre el que escribir.

---

## RETURN VALUES

Nada
