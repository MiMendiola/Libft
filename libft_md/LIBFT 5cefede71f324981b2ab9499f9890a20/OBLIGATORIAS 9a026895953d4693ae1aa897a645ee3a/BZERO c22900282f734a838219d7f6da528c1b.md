# BZERO

---

## SYNOPSIS

```c
#include <string.h>

void bzero(void *s, size_t n);
```

---

## DESCRIPTION

The `bzero()` function writes n zeroed bytes to the string s. If n is zero, `bzero()` does nothing.

Writes the len that you say in the strings with \0.
